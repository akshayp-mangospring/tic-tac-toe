**This is a simple Tic-Tac-Toe game built with vanilla HTML, CSS, JS which you can play in the browser on all devices**

You can play the game on: https://akshayp-mangospring.github.io/tic-tac-toe

## For local setup:

## Docker Way:
### Prerequisites:
- Install docker from https://docs.docker.com/engine/install/

#### In terminal:
```bash
docker compose up
```

---

## Manual Way:
### Prerequisites:
- NodeJS LTS
- A latest web browser(Chrome/Safari/Firefox/Edge)

### To install deps

#### In server:
```bash
cd server
npm install
```
#### In client:
```bash
cd client
npm install
```

### To start

#### In server:
```bash
cd server
npm start
```
#### In client:
```bash
cd client
npm start
```

Start playing locally on: **http://localhost:3000**

## For Testing:
### Prerequisites:
- Ubuntu:
```bash
apt-get install libgtk2.0-0 libgtk-3-0 libgbm-dev libnotify-dev libgconf-2-4 libnss3 libxss1 libasound2 libxtst6 xauth xvfb
```
- NodeJS LTS
- A latest web browser(Chrome/Safari/Firefox/Edge)

### To run tests
```bash
npm run test
```
